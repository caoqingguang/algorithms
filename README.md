## 01 背包问题

**描述**：
```
三块石头  
A 重量 5  价值 20
B 重量 4  价值 10
C 重量 3  价值 12

背包容量 10，怎么装石头，背包价值最大？
```

计算过程见**code**，计算**结果**如下：
```
最大价值：32, 负重：8 
详情：[A->{weight:5,price:20}, C->{weight:3,price:12}]
```
